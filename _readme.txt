This it he base directory of the mismo-bindings project. The goal of this
project is to create Java (and other platforms) bindings of MISMO XML
documents. See the doc folder for more.

Project Release Version: See "project.release.version property" in project.properties file
Project WebSite: http://code.google.com/p/mismo-bindings/

Structure:
doc: documentation
src: source files
src/java: Java source files
lib/java: Java libraries needed at runtime
tools: Tools needed at build-time
